# VexTweaks Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- Initial release of VexTweaks
- Modern dark-themed WPF interface
- Sidebar navigation with 10 sections
- Dashboard with system information and optimization score
- 25+ system optimization tweaks across multiple categories
- Performance optimizations (7 tweaks)
- Gaming optimizations (7 tweaks)
- Network optimizations (3 Pro tweaks)
- Privacy controls (4 tweaks)
- Hardware-bound license system with Free/Pro tiers
- Three built-in optimization profiles (Performance, Gaming, Streaming)
- Profile export/import functionality
- Automatic tweak backup and revert support
- Action logging system
- Admin elevation with UAC prompt
- Category-based tweak filtering
- Animated toggle switches
- Tweak tagging system (Recommended, Advanced, May Reduce Features)
- Real-time tweak status indicators
- System information display (CPU, RAM, GPU, Windows version)
- Optimization score calculation

### Features by Category

#### Performance
- Disable Visual Effects
- Disable Transparency
- Optimize Processor Scheduling
- Remove Startup Delay
- Optimize Memory Management
- Disable Prefetch/Superfetch
- Disable Windows Search Indexing

#### Gaming
- Disable Game Bar
- Disable Game DVR
- Gaming Priority Mode
- Hardware-Accelerated GPU Scheduling
- Disable Fullscreen Optimizations
- Disable Mouse Smoothing

#### Network (Pro)
- Reduce Network Throttling
- Disable Nagle's Algorithm
- Optimize Network Adapter

#### Privacy
- Disable Telemetry
- Disable Activity History
- Disable Location Tracking
- Disable Advertising ID

### Technical
- Built with .NET 8.0 and WPF
- MVVM architecture with dependency injection
- CommunityToolkit.Mvvm for ViewModels
- System.Management for hardware information
- Registry-based tweak engine
- Comprehensive error handling and logging

### Documentation
- Comprehensive README with features and usage
- Build and deployment instructions
- Contributing guidelines
